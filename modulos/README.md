# Módulos de la interfaz

Esta carpeta contiene los módulos llamados desde `main.py`. Incluye implementaciones mínimas
para que la app abra las ventanas sin error. Reemplaza la lógica con tu flujo real.

## Estructura
- `modulo1_cargador/gui_cargador.py`: ventana para cargar archivos `.tex`.
- `modulo2_auditor/gui_auditor.py`: ventana de auditoría híbrida.
- `modulo2_auditor/gui_teoria.py`: carga de teoría/base de conocimiento.

Asegúrate de mantener los `__init__.py` si usas imports relativos.
